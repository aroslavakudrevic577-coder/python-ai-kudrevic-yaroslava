# Dataset Examples

## Примеры данных / Data Examples

В этой папке находятся примеры небольших наборов данных для обучения и тестирования аналитических навыков.

This folder contains sample datasets for learning and testing analytical skills.

### Файлы / Files:

- **example_economy.csv** - Данные по экономическим показателям / Economic indicators data
  - Год / Year
  - ВВП / GDP
  - Инфляция / Inflation
  - Безработица / Unemployment

- **example_sales.csv** - Данные по продажам / Sales data
  - Дата / Date
  - Товар / Product
  - Количество / Quantity
  - Цена / Price

- **example_students.csv** - Данные по студентам / Student data
  - Имя / Name
  - Возраст / Age
  - Оценка по математике / Math grade
  - Оценка по программированию / Programming grade
